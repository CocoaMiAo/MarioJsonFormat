###使用介绍
- 可以解析Json以及XML类型的文本
- 依据个人需求选择是否生成swift文件,默认生成
- 在**Root class name** 文本框中输入想要生成的类文件名称,默认**Mario**
- 在**Classes prefix** 文本框中输入需要展示的前缀,默认为空
- 点击**Save**按钮保存文件的前提是点击**Format**格式化生成类文件内容
